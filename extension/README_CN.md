# Shorts Tracker - 你的短视频时间管理助手 🟢

**拒绝无意识的刷屏，掌握你的数字生活。**

Shorts Tracker 是一款专为 YouTube Shorts 和 TikTok 设计的浏览器扩展，旨在帮助你量化观看习惯，提供实时反馈，并在你沉迷时温柔提醒。它不仅仅是一个计数器，更是你对抗算法黑洞的理性伙伴。

## ✨ 核心功能

### 1. **全自动追踪 (Smart Tracking)**
- **无感运行**：无需手动开启，当你进入 YouTube Shorts 或 TikTok 页面时自动激活。
- **精准统计**：实时记录你观看的**视频数量**和**总时长**。
- **双平台支持**：完美兼容 YouTube 和 TikTok。

### 2. **实时悬浮胶囊 (The Capsule)**
- **沉浸式显示**：一个精美的小型悬浮窗（Capsule）常驻屏幕角落。
- **即时反馈**：每刷一个视频，数字即刻跳动，直观地告诉你：“这是你今天看的第 N 个视频”。
- **可配置**：如果你想专注于内容，可以在设置中一键隐藏。

### 3. **防沉迷提醒 (Reality Check)**
- **温柔打断**：每观看 **15 个视频**，插件会弹出一个非侵入式的提醒。
- **健康建议**：提示你休息一下眼睛，喝口水，或者意识到时间的流逝。

### 4. **强大的数据仪表盘 (Dashboard)**
- **数据可视化**：提供过去 7 天的详细观看数据柱状图。
- **每小时详情**：点击图表上的任意一天，即可查看当天的**每小时分布图** (Hourly Breakdown)，了解你在哪个时间段最容易沉迷。
- **Liquid Glass UI**：独创的“液态玻璃”实验性界面，带来极致的视觉体验（可在设置中开启）。

### 5. **隐私至上 (Privacy First)**
- **本地存储**：所有数据仅保存在你的浏览器本地 (Chrome Storage)，**绝不上传**到任何云端服务器。
- **数据掌控**：随时可以在设置中一键清空所有历史记录。

---

## 🚀 如何使用

1. **安装插件**：加载 Shorts Tracker 到你的 Chrome/Edge 浏览器。
2. **开始浏览**：打开 YouTube Shorts 或 TikTok，你会看到左上角出现绿色的计时胶囊。
3. **查看详情**：点击浏览器工具栏的插件图标，查看今日概览；点击“更多设置 (Gear Icon)”进入完整仪表盘。

---

## 🎨 个性化体验

在设置页面中，你可以：
- 开启/关闭 **实验性 UI 模式**（推荐尝试，非常酷炫的深色磨砂玻璃质感）。
- 开启/关闭 **悬浮胶囊**。
- 重置所有统计数据。
